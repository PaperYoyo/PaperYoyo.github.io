Tools I use for music mixing

  The church I go to I have been privileged enough to help with sound, lights, recording, and producing the services and podcasts that can be found of the Mercy Hill website. For the mixing we us a bringer x-32 compact mixing board connected to a laptop that records through a program called reaper. We use amazon S3 bucket hosting to make sure downloading and streaming of the podcasts from the website is as fast and as smooth as possible.

Main commands Rebasing Pushing
git clone string_url: clones git repository git fetch --all: fetches all changes from remote branches git add . : adds changes to commit
git status: shows the current changes made to the current branch git rebase remote_branch(in our case origin/master): applies latest changes from remote branch to current local branch git commit : commits the current branch's changes
Hi!!! git push website HEAD: Publishes commit to website git push origin HEAD: pushes commit to the repository's remote origin's HEAD
